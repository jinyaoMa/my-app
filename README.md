# my-app starter with wails v3

prepare for running, install node24 and pnpm, then run below

```bash
pnpm install
go get -u -t ./... # change github.com/wailsapp/go-webview2 to v1.0.21
go mod tidy
```

try to run app

```bash
pnpm app:dev
```
